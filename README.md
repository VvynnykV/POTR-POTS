- POTR Pots — Eco-Friendly Self-Watering Planters.
  This project is a modern e-commerce platform designed for POTR, a brand that reimagines plant care through sustainability and innovation. The website showcases origami-inspired, self-watering plant pots, focusing on eco-conscious consumers and sleek, functional interior design.

- Technologies Used
  Core
  * HTML5 – Semantic markup for a structured and accessible web experience.
  * CSS / SCSS – Advanced styling using SASS (via @parcel/transformer-sass) for modular and maintainable styles.
  JavaScript (ES6+) – Logic and interactivity implemented with modern JavaScript standards.

  Build & Bundling
  * Parcel (v2.12.0) – A zero-config build tool used for bundling assets and lightning-fast development.
  * Scripts – Custom development environment for building, testing, and deploying the project.
  * Gulp-based Boilerplate – Optimized workflow for frontend layout development.

  Code Quality & Tooling
  * ESLint – Ensuring high-quality JavaScript code and adherence to best practices.
  * Stylelint (v16.7.0) – A mighty linter that helps avoid errors and enforces consistency in SCSS files.
  * LintHTML – Tool for maintaining clean and valid HTML structure.
  * Prettier – An opinionated code formatter for consistent styling across the entire codebase.

  Deployment
  * GitHub Pages – Automated deployment processes integrated into the development workflow.

- Features
  * Responsive & Adaptive Design: Precise implementation of layouts for mobile (320px), tablet (768px), and desktop (1440px), ensuring a seamless experience across all screen sizes.
  * Modular SCSS Architecture: Styles are organized using modern SCSS features (variables, mixins, nesting) for high maintainability and scalability.
  * Semantic HTML5: Built with accessibility (A11y) in mind, using semantic tags to ensure the site is SEO-friendly and screen-reader accessible.
  * Interactive UI Components:
    -- Custom Sliders/Carousels: For showcasing product galleries and origami assembly steps.
    -- Mobile Navigation: A smooth, functional hamburger menu for smaller screens.
  * BEM Methodology: Clean and consistent naming conventions for CSS classes to avoid style conflicts.
  * Cross-Browser Compatibility: Optimized for the last two versions of all major browsers (Chrome, Firefox, Safari, Edge).
  * Fast Performance: Zero-config asset optimization and bundling provided by Parcel, ensuring quick load times.
